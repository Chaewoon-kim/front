* 자바스크립트의 특징
1. 조각난 소스 코드의 형태로 HTML 페이지에 내장
2. 별도의 컴파일 과정 없이 바로 실행
3. 자바스크립트는 C언어 구조를 차용하고 단순화시킴

*개발자 모드 콘솔에서 바로 자바스크립트 코드를 연습해 볼 수 있음


(1)변수 선언 
-> let : 지금부터 ...을 만들겠습니다. 변수 선언 위한 함수 (= var)
    ex) let a = 3, a에 3이라는 값을 넣음
-> 증감 연산자(++,--)
    ex) a++ 후위 증감 , 답이 나중에 나옴
        ++a 전위 증감, 먼저 계산하기 때문에 콘솔 창에 계산 값이 바로 나옴
let 대신 var로 변수 선언하자.
*let 과 var의 차이
-> var은 값을 중복해서 선언 가능
-> let은 한 번 초기화하면 재선언 불가


*변수 이름 짓기
1. 의미 있는 단어 사용 (예약어 x)
2. 낙타 표기법 관례 (addNum , javaScript, ...)


*String (문자열)
*Number (숫자형)
*Boolean(참 또는 거짓)

(2)여러 함수들
[콘솔]에서 해당 변수를 보기 위해서는 console.log()함수 사용
[실제 화면]에 나타내고 싶을 때는 document.write()함수  (권장x)
[변수의 타입]을 알고 싶을 때는 conosle.log(typeof())함수 사용
[프롬프트]에 입력을 하고 싶다면 prompt()함수 사용

(3)변수 선언
const : 변하지 않는 상수
var: 변수 선언
let: 재선언 불가.
